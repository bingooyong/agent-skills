## 2. 错误码约定

| 码   | 含义     | 处理建议     |
|------|----------|--------------|
| 400  | 参数错误 | 校验请求体   |
| 401  | 未认证   | 检查 Token   |
| 404  | 资源不存在 | 检查 ID 等  |
| 5xx  | 服务异常 | 重试/联系运维 |
